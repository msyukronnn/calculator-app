# 🧮 Modern Calculator App

Aplikasi kalkulator berbasis web yang responsif, ringan, dan akurat. Dibangun dengan fokus pada pengalaman pengguna yang intuitif dan desain yang bersih.

---

## ✨ Fitur Utama

- **Operasi Aritmatika:** Penjumlahan, pengurangan, perkalian, dan pembagian.
- **Floating Point:** Mendukung perhitungan desimal dengan presisi tinggi.
- **Responsif:** Tampilan menyesuaikan secara otomatis untuk HP, Tablet, dan Desktop.
- **Tema Gelap:** Nyaman di mata untuk penggunaan jangka lama.
- **Fungsi Hapus:** Dukungan untuk *Clear Entry* (CE) dan *All Clear* (AC).

## 🛠️ Teknologi yang Digunakan

Proyek ini menggunakan *stack* teknologi standar web untuk memastikan performa maksimal:

* **HTML5** - Struktur aplikasi.
* **CSS3** - Custom properties dan Flexbox/Grid.
* **C#** - Logika kalkulasi.
